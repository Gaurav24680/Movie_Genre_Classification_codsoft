## 📌 Project Overview

Movie Genre Classification is a Machine Learning project that predicts the genre of a movie based on its text description or plot summary. The system uses Natural Language Processing (NLP) and Machine Learning algorithms to analyze movie descriptions and classify them into genres such as Action, Comedy, Drama, Horror, Thriller, and more.

This project demonstrates the practical application of text classification, data preprocessing, feature extraction, and supervised machine learning techniques.

---

## 🎯 Objective

The main objective of this project is to:

- Classify movies into appropriate genres
- Apply Natural Language Processing techniques
- Train and evaluate machine learning models
- Improve prediction accuracy using text feature extraction methods
